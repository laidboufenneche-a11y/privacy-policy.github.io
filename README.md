# Social Networks ID Extractor – Privacy Policy

**Last updated:** March 24, 2026

## Overview
Social Networks ID Extractor is a Chrome extension designed to extract publicly visible profile information from social media platforms. The extension operates entirely within the user’s browser and only functions when explicitly triggered by the user.

---

## 1. Data Collection
The extension collects data **only when the user manually enters a profile URL and clicks "Get Account Data"**.

The following publicly available information may be extracted:
- **Account ID** – Unique identifier of the profile  
- **Full Name** – Displayed name of the profile owner  
- **Profile URL** – Link to the profile  
- **Gender** – If publicly available  
- **Additional Metadata** – Any other publicly visible profile details  

---

## 2. How Data is Processed
- All processing occurs **locally in the user's browser**
- Data extraction happens **only after explicit user action**
- **No data is sent to external servers**
- **No data is stored permanently**
- **No data is shared with third parties**
- Extracted data is **temporary** and disappears when the extension popup is closed

---

## 3. Permissions Used
The extension requires the following permissions strictly for functionality:

- **activeTab**  
  Accesses the current tab only when the user activates the extension  

- **scripting**  
  Injects scripts to extract data from the page on user request  

- **Host permissions**  
  Limited to:
  - facebook.com  
  - instagram.com  
  - twitter.com  
  - tiktok.com  

---

## 4. Data Sharing
No data is:
- Shared  
- Sold  
- Transferred  

All operations remain strictly on the user's device.

---

## 5. User Rights
- No personal data is stored or retained  
- Full control remains with the user at all times  
- No tracking or background data collection occurs
